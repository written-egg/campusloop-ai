# 国内部署说明

这个项目是 Node.js 服务，不是纯静态网页。要生成中国大陆可访问的网址，需要部署到一台国内服务器，例如腾讯云轻量应用服务器、阿里云 ECS、华为云云耀服务器。

## 需要准备

- 一台国内云服务器，推荐 Ubuntu 22.04。
- 安全组开放端口：`80`、`443`，演示阶段也可以临时开放 `5173`。
- DeepSeek API Key 只设置在服务器环境变量里，不要写进前端代码。

## 方式一：直接运行 Node

上传项目到服务器后执行：

```bash
cd ai-campus-secondhand-workbench
export PORT=5173
export DEEPSEEK_API_KEY=你的DeepSeekKey
export DEEPSEEK_MODEL=deepseek-chat
node server.js
```

浏览器访问：

```text
http://服务器公网IP:5173
```

## 方式二：Docker 运行

```bash
cd ai-campus-secondhand-workbench
docker build -t campusloop-ai .
docker run -d \
  --name campusloop-ai \
  -p 5173:5173 \
  -e PORT=5173 \
  -e DEEPSEEK_API_KEY=你的DeepSeekKey \
  -e DEEPSEEK_MODEL=deepseek-chat \
  campusloop-ai
```

浏览器访问：

```text
http://服务器公网IP:5173
```

## 绑定正式域名

如果有域名，建议用 Nginx 把 80 端口转发到 5173：

```nginx
server {
  listen 80;
  server_name 你的域名;

  location / {
    proxy_pass http://127.0.0.1:5173;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
  }
}
```

之后访问：

```text
http://你的域名
```

## 数据库说明

当前演示版使用 `data/db.json` 存储用户和商品，适合课程演示。服务器部署后，这个文件会保存发布的商品和登录用户。后续如果要做正式平台，再迁移到 MySQL 或 MongoDB。
